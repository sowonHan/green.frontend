함수라든지 코드 전체를 객체화하면 계속 가져다 쓸 수 있어서 재사용성과 생산성이 높아진다.

객체 : 기능(Method = 함수(function)), 속성

Java에서는 메소드라고 부르고 Javascript에서는 함수라고 부른다.

객체: 기능(볼륨 증감, 키, 끄고 켜기) & 속성(너비, 색상, 해상도)

ex) 자동차: 기능(주행, 에어컨, 음악 등) & 속성(색상, 연비, 브랜드)

ex) 인간: 기능(걷다, 달리다, 잠) & 속성(키 160 나이 30)

교제 88p 예시 참고

자바스크립트에서 제공하는 객체

내장 객체, 브라우저 객체(BOM), 문서 객체(DOM)

내장 객체 : 자바스크립트 엔진에 내장된 객체. (자바스크립트 만든 사람이 미리 만들어둔 거라고 보면 됨) Date(날짜), Math(루트 같은 수학 함수), Array(데이터를 편하게 다루게 해줌), String(문자열 다루기 쉽게)

브라우저 객체(BOM) : (window.)prompt("",""), (window.)location.href, alert("")

브라우저 객체는 브라우저에서 지원해주는 것

문서 객체(DOM) : html 문서와 태그. html에서 배운 것들의 구조를 DOM TREE라고 보면 됨
