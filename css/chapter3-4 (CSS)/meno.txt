herf가 속성, "#"이 속성값

data- 를 붙이면 커스터마이징으로 속성을 부여할 수 있다.

input은 아무것도 지정 안 할 경우 디폴트 타입이 text

CSS 크기 단위
절대 크기 : px (pixel 픽셀) - 아무것도 입력 안 하면 브라우저의 디폴트 크기로 나온다.
상대 크기
1) % - 원래(디폴트) 크기의 비율로 조절됨. 레이아웃 지정 시 주로 사용
2) em - 배율. 가독성이 퍼센트보다 좋아서 더 많이 쓴다. 부모 태그가 폰트 사이즈 기준
3) rem - 기능은 em과 똑같이 배. html 페이지 폰트 사이즈가 기준

경로
(1) 절대 경로 : 절대 위치라는 의미. 위치 전체 표기. 코딩할 때는 99% 정도 쓰지 않는다.
                예시] C:\project\green.frontend\html\chapter1-2 (HTML)\0228_1.HTML
                서버라든지 OS라든지 제한이 너무 많음
(2) 상대 경로
    1) ./ - 현재 폴더부터. 같은 폴더에 있을 경우
    2) ../ - 이전 폴더
