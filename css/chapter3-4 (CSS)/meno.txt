태그에는 속성과 속성값이 있고 이건 정해져있다
<a href = "#"></a>
herf가 속성, "#"이 속성값

data- 를 붙이면 커스터마이징으로 속성을 부여할 수 있다.

input은 아무것도 지정 안 할 경우 디폴트 타입이 text

**CSS 크기 단위**
절대 크기 : px (pixel 픽셀) - 아무것도 입력 안 하면 브라우저의 디폴트 크기로 나온다.
상대 크기
1) % - 원래(디폴트) 크기의 비율로 조절됨. 레이아웃 지정 시 주로 사용
2) em - 배율. 가독성이 퍼센트보다 좋아서 더 많이 쓴다. 부모 태그가 폰트 사이즈 기준
3) rem - 기능은 em과 똑같이 배. html 페이지 폰트 사이즈가 기준

**경로**
(1) 절대 경로 : 절대 위치라는 의미. 위치 전체 표기. 코딩할 때는 99% 정도 쓰지 않는다.
                예시] C:\project\green.frontend\html\chapter1-2 (HTML)\0228_1.HTML
                서버라든지 OS라든지 제한이 너무 많음
(2) 상대 경로
    1) ./ - 현재 폴더부터. 같은 폴더에 있을 경우
    2) ../ - 이전 폴더

**Position 메모**
    static : position의 디폴트값. 원래대로 돌릴 때 사용할 수 있다. (상대 위치)
    fixed는 스크롤이 있어도 고정되는 것. 메뉴에 주로 쓰인다.
    z-index는 수치가 클수록 앞으로 온다. 마이너스도 가능하다. position과 flex-item에서만 적용 가능.

******CSS 우선순위***** 
    ☆ css는 같은 속성이 여러 개가 있을 경우 가장 마지막에 오는 걸 우선 적용시킨다.

    **선택자 우선순위**
    id 선택자 > class 선택자 > 태그 선택자

    **CSS 스타일**
    1) inline 스타일 : 태그에 style="" 직접 입력하는 방법.
    2) 내부 스타일 : head에 style 태그
    3) 외부 스타일 : css파일을 불러오는 것. Emmet으로 link css 하면 자동완성 된다.
    우선순위 - 인라인 > 내부 > 외부 > 브라우저 기본 값

    !important : 다 무시하고 최우선으로 적용시키는 것.
        예시) p {color : blue !important;}

    전체 태그 *와 해당 태그 적용 시 : 해당 태그 > 전체 태그
